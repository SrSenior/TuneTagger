# Development Notes

This document will be used as a personal development log for the project.

Since this is my first project using .NET, the content of this document will evolve alongside the repository. With each relevant commit, I will document the features implemented, the decisions made, and the concepts I learn or revisit during the development process.

The main purpose of this document is not only to keep a record of the project’s progress, but also to create a self-study resource that I can revisit in the future whenever I need to review specific concepts, commands, errors, or implementation details.

The language used throughout this document will not necessarily be highly technical. However, it will aim to remain clear, organized, and professional enough to be useful both as personal documentation and as a learning reference.

Similarly, a Spanish version of this document will be maintained for my own reference, while this English version will be available for any reader who may prefer or need to consult the documentation in English.
